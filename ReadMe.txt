Взлом ПИН кода для игры "Пандемия: Дорога домой"
=============================

Спасибо, что решили воспользоваться моим приложением.


Установка
------------

Установка не требуется, программа представляет из себя консольное приложение.
Собранная и готовая к работе версия находится в директории Release\DigitalLock.exe.


Использование
-----------

После запуска приложения откроется консольное окно с текстом:

Now PIN count: 1680 <--- количество оставшихся ПИН кодов
Try enter PIN: 3617 <--- ПИН код который необходимо ввести в игре
Enter response:     <--- Поле для ввода ответа дешифратора (из игры)

Вам будет предложен ПИН код, который необходимо ввести в игре, ответ дешифратора необходимо ввести в приложение.
После этого будет произведен отсев ПИН кодов, которые не соответствуют текущему ответу и предложен новый ПИН код.
Через несколько итераций (обычно от 3 до 6) программа подберет для Вас верный ПИН код загаданный игрой.
